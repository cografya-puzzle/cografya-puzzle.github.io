# Coğrafya Bulmacaları

Lise coğrafya derslerine yönelik interaktif çapraz bulmaca oyunu. Türkiye coğrafyası, iklim, kayaçlar, nüfus ve daha fazlasını kapsayan 79 bulmaca içerir.

## Oyna

**https://cografya-puzzle.github.io**

## Özellikler

- 79 bulmaca, birden fazla bölüm
- İpucu sistemi (bulmaca başına 3 hak)
- Anlık puanlama
- Mobil uyumlu arayüz
- Bölüm haritası ile kolay gezinme

## Kapsanan Konular

- Harita bilgisi
- Türkiye'nin yer şekilleri
- İklim bilgisi
- Kayaçlar ve iç kuvvetler
- Su kaynakları
- Toprak türleri
- Nüfus

## Teknik

Saf HTML/CSS/JavaScript, herhangi bir framework veya backend bağımlılığı yok. GitHub Pages üzerinde barındırılmaktadır.
